# YRelaxedSku - 京东商品好评率查询工具

## 📌 项目简介
YRelaxedSku是一款自动化获取京东(JD)商品好评率的工具，帮助用户快速批量查询商品评价数据。

## ✨ 功能特性
### 核心功能
- 📥 批量导入SKU列表
- 🔍 一键查询商品好评率
- 📤 导出Excel格式结果报表

### 高级设置
- ⏱️ 自定义请求延时
- 🏷️ 按商品品类筛选
- 🔄 自动重试机制
- 🚀 多线程并发处理
